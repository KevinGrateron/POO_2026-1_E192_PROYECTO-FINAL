# 🚀 Descripción del Proyecto

# 

# Este proyecto fue desarrollado en Java utilizando Swing y Sqlite como base de datos.

# 

# El sistema permite gestionar información empresarial mediante módulos CRUD conectados directamente a una base de datos relacional.

# El proyecto implementa conceptos de:

# Programación Orientada a Objetos

# Arquitectura por capas

# Conexión JDBC

# Interfaces gráficas con Swing

# Gestión de eventos

# CRUD completo

#
